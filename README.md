## 👋 About Me

<img align="right" width="360" src="https://cdn.dribbble.com/users/1162077/screenshots/3848914/programmer.gif" />

Hi, I’m **Harun Ar Rashid** — a passionate **Full Stack MERN Developer** from **Dhaka, Bangladesh** 🇧🇩.

I build **modern, scalable, responsive, and user-focused web applications** using the MERN stack, Next.js, and clean UI architecture.  
My goal is simple: **turn real-world ideas into polished digital products** that feel smooth, useful, and production-ready.

- 🚀 Currently building and improving **ShebaSathi**
- 🧠 Learning **Advanced Next.js, Backend Architecture & System Design**
- 💼 Interested in **SaaS, CRM, E-commerce, Healthcare Tech & Service Platforms**
- 💬 Ask me about **React.js, Next.js, Node.js, Express.js, MongoDB & REST APIs**
- ⚡ I care about **clean code, performance, responsive design, and product experience**
